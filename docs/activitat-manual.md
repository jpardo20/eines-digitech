# Activitat manual instal·lació programes portàtil centre

Cal que fent servir totes les imatges que trobareu dins de la carpeta [**`img`**](../img/) d'aquest repostori, creeu un manual amb tots els passos que cal fer per instal·lar i configurar totes les aplicacions que calen per realitzar el cicle.

Els passos son els següents:

1. Creació d'un nou usuari local al vostre portàtil

1. Configuració de l'explorador de fitxers de windows per mostrar les extensions dels fitxers i els fitxer ocults.

1. Instal·lar Virtual Box

  [VC_redist.x64.exe](https://aka.ms/vs/17/release/vc_redist.x64.exe)

  [VirtualBox-7.2.2-170484-Win.exe](https://download.virtualbox.org/virtualbox/7.2.2/VirtualBox-7.2.2-170484-Win.exe)

1. Instal·lació de Python 3

  [Latest Python 3 Release - Python 3.13.7 (python-3.13.7-amd64.exe)](https://www.python.org/ftp/python/3.13.7/python-3.13.7-amd64.exe)

1. Instal·lació git

  [Click here to download the latest (2.51.2) x64 version of Git for Windows.](https://github.com/git-for-windows/git/releases/download/v2.51.2.windows.1/Git-2.51.2-64-bit.exe)

1. Instal·lació Visual Studio Code

 [Download Visual Studio Code] (https://code.visualstudio.com/sha/download?build=stable&os=win32-x64-user)

1. Configuració de Visual Studio Code

  1.1. Configurar Auto save per què es guardin automàticament els fitxers
  1.2. Instal·lar el plugin del "Live Server" 
  1.3. Instal·lar el plugin del "Pylance"
  1.4. Configurar el Terminal perquè sempre obri el Terminal com **`bash`**
  1.5. Configurar el vostre usuari de github

1. Instal·lació Cisco Packet Tracer

  Para obtener e instalar su copia de Cisco Packet Tracer, siga las instrucciones del siguiente enlace: [Cisco Packet Tracer](https://www.netacad.com/resources/lab-downloads)
